<strong style="font-size: 20px;">1. 游戏简介</strong>

《科技之城》是一款融合了舰船收集、卡牌培养和合作竞争的游戏。玩家扮演一名未来科技专家，需要通过获取基础资源、购买突破卡牌、兑换项目卡牌和建造建筑来获取研究点数。在游戏中，玩家可以与其他人合作建造建筑，并协商资源的归属。当所有项目完成且建筑建造完毕时，游戏结束，拥有最多研究点数的玩家获胜。

<strong style="font-size: 20px;">2. 背景介绍</strong>

在遥远的未来，人类面临严重的土地和食物短缺。为了生存，人们逐渐机械化自身，以资源自给自足，从而诞生了一种新的族群——“科技族”（Techs）。科技族通过研究技术、建造建筑并积累研究点数来提升自身优势，以在竞争中击败对手。

1. 将高级建筑卡、积分建筑卡、技术项目卡、技术突破卡分别洗牌，并按顺序堆放。
   
2. 翻开一张高级建筑卡，四张积分建筑卡，五张技术项目卡和技术突破卡。
   
3. 每位玩家抽取一张角色卡作为自己的角色。
   
4. 玩家决定谁先开始游戏。
   
5. 按顺时针方向分配金币：
   
  2 人游戏，前一个玩家分配 1 金币，后一个玩家分配 2 金币；
  
  3 人游戏，前两个玩家各分配 1 金币，最后一个玩家分配 2 金币；
  
  4 人游戏，前两个玩家各分配 1 金币，后两个玩家各分配 2 金币。

<strong style="font-size: 24px;">玩家目标</strong>

<strong style="font-size: 24px;">结束条件</strong>

所有项目和建筑完成时，游戏结束。

<strong style="font-size: 24px;">胜利条件</strong>

<strong style="font-size: 24px;">游戏节奏</strong>

游戏的整体节奏分为两个阶段：

1. **准备阶段：**主要集中在资源收集，玩家了解要兑换的项目卡，并努力收集尽可能多的资源。

2. **互动阶段：**主要集中在合作与竞争，玩家在获得一定数量的项目卡后，合作以赚取更多积分。

<strong style="font-size: 24px;">数值设计</strong>

游戏的核心是资源交换和合作竞争，因此数值设计围绕**资源交换**展开。

<strong style="font-size: 20px;">1. 资源分级</strong>

<strong style="font-size: 20px;">2. 转换公式</strong>

**LV.1 资源** → **金币资源**
   
**金币资源** → **高级建筑**  

**LV.1 资源 + 高级建筑** → **LV.2 资源**
 
**LV.1 资源** → **LV.3 资源**  

**LV.1 资源 + LV.2 资源** → **LV.4 资源**

**LV.3 资源 + (LV.4 资源)** → **LV.5 资源**  

**(LV.4 资源)** → **点数**

**(LV.5 资源)** → **点数**  

**(金币资源 + LV.5 资源 + LV.4 资源)** → **点数建筑**  

**点数建筑** → **点数**

**LV.5 资源数量达到最大值** → **点数**  

**建筑资源数量达到最大值** → **点数**

<strong style="font-size: 20px;">3. 数值常数</strong>

| 列名 | 描述 | 值 |
|------|------|----|
| A    | 基础系数 | 1 |
| B    | 高级资源转换系数 | 0.2 |
| C    | 建筑系数 | 3 |
| D    | 价格 : 点数 | 4 |
| E    | 回合数 : 点数 | 4 |
| F    | 每回合基本资源预期 | 2 |
| G    | 系数（Lv5资源与点数建筑） | 0.75 |
| H    | 系数（Lv4资源与点数建筑） | 0.45 |
| I    | 点数建筑点数 | 10 |


<strong style="font-size: 20px;">3. 数值公式</strong>

①**LV.1 资源**:  

   价格 = A

②**高级建筑**:  

   价格 = A * C

③**LV.2 资源**:  

   价格 = LV.1 资源价格 + 高级建筑价格 * B

④**LV.3 资源**:  

   价格 = 所需资源数量 * 价格的累加

⑤**LV.4 资源**:  

   价格 = 所需资源数量 * 价格的累加  

   点数 = 价格 / D

⑥**LV.5 资源**:  

   价格 = 所需资源数量 * 价格的累加  
   
   回合数 = 所需资源数量的累加 / F  
   
   点数 = 价格 / D + 回合数 / E

⑦**点数建筑**:  

   点数 = I  
   
   价格 = 点数 * D

⑧**所需 LV.5 资源数量** = 1

⑨**所需 LV.4 资源数量** =  

   [价格 - 所需 LV.5 资源数量 * LV.5 资源价格 * G] / (LV.4 资源价格 * H)
